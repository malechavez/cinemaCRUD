# cinemaCRUD

Sistema de Gestión de Películas (CRUD en Python)

Este proyecto consiste en una aplicación de consola desarrollada en Python que permite gestionar una cartelera de películas utilizando una base de datos SQLite.

🚀 Funcionalidades

➕ Agregar películas
🔍 Buscar películas por nombre o género
📄 Ver listado de películas (ordenadas por distintos criterios)
✏️ Modificar datos (nombre, género, plataforma, idioma)
❌ Eliminar películas
🗑️ Vaciar base de datos
📊 Consultas de cantidad de películas por género y plataforma
🛠️ Tecnologías utilizadas
Python
SQLite (base de datos)
colorama (para interfaz en consola)

🧠 Conceptos aplicados
Operaciones CRUD (Create, Read, Update, Delete)
Manejo de bases de datos con SQLite
Consultas SQL (SELECT, INSERT, UPDATE, DELETE, GROUP BY)
Validación de datos de entrada
Organización del código en funciones


▶️ Cómo ejecutar el proyecto
Clonar el repositorio:
git clone https://github.com/malechavez/cinemaCRUD.git
Ejecutar el archivo principal:
python main.py
🗄️ Base de datos

El proyecto utiliza una base de datos SQLite.
La base se crea automáticamente al ejecutar el programa si no existe.
